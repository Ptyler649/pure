March 2014 News

27/03/2014 - Been working hard on the day job and so not much time over the last week, however just managed to get the JSImongo alpha code up to git hub for the first time. Also posted this week some HOW TO notes on setting up an AWS Cloud web server that may help some of the community looking to do this for the first time. The HOW TOO's are off the home page. Looking to get sample code posted here that is more related to the myJarvis piece over the next week and this will post tweets up into mongoDB.

18/03/2014 - This week I have de-coupled both the day time element and the delay element from the code and added these to the jsiconfig.txt file to allow for easy modification. Edit the 4th and 5th line of the config file to adjust accordingly. Essentially 900 here equates to 960 mins if the delay is written in as 60000ms (60 seconds) and would give a realtime trading day of 16hrs. As this is a simulation you can move this write down do a 1ms delay that depending on network and cpu spped will give you the fastest results but will randomise the date time selection written in as per the JSON. Keep in mind the number of days you have also configured here and if inserting to mongo the network speed will effect performance.

12/03/2014 - Fixed some issues with the website here, added the follow twitter button and the link to mongodb labs.

12/03/2014 - This week I have cleaned up the code a little and added a couple of functions. First I have added the dates and entities functionality, this allows the simulation of multiple dates simply by amending the dateslist.txt file. In addition the entities funcitonality gives simulations of products being sold across mutiple entities / shops / offices etc. I have also added the -i and -d commands when you execute jsimongo. -i disables the insertion into mongoDB to allow for testing. -d allows you to disable the display function to the console. Without these set the default will be to insert and display the transactions.
