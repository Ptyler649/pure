Products

MEET - a java based app that brings together mongoDB and Twitter and shows a simple way to start to insert your company tweets up into mongoDB, this then allows for map reduce queries, business intelligence and marketing functions to take place that just simple could not be done before the availability of these technologies.

MEET functionality includes: 20 tweets per call are inserted from your nominated twitter account into your chosen mongoDB Database allowing for further analysis.

MYJARVIS - a console based application written in java that using a artifical intelligence style interface controls and displays bespoke messaging, myJarvis allows the user to control all aspects of there life in a CLI (Complete Life Interface)

myJarvis functionality includes: AI interface, twitter, facebook and email feeds, weather and stock reports, note taking, appointment reminders, remote appliance controls.

JSIMONGO - a java 1.7.0.51 program that uses the mongo java API to post documents up into a MongoDB Cloud instance in a time lapsed manner, this significantly helps in the test, development and visualization of many different types of realtime interfaces

JSImongo functionality will include: Run as a service, sample JSON, user defined random member data and data profiling. Full flexibility is the target. Note JSI will need to be run external to any firewalls - common place in a standard office LAN environment. This is very early days but an alpha version is now available on the downloads page.
